# Summary-Generation-Web-App
Summary Generation Web App
Here are the steps to execute the Project- Summary Generation web app using 
NLP . 
1. Choose an Integrated Development Environment (IDE): 
Both VS Code and PyCharm are excellent choices. Choose the one you're 
more comfortable with. 
2. Install VS Code or PyCharm: 
Follow the provided links (Vs Code Link, PyCharm Link) to download and 
install your chosen IDE. 
3. Install Python: 
If you haven't installed Python, follow the Python Link to download and 
install Python 3.6 or a higher version. 
4. Project Setup: 
Open your chosen IDE and create a new project for web app. 
5. Install Required Libraries: 
Open the terminal within your IDE and install the required libraries using 
the provided commands: 
Copy this codes in terminal and install 
 pip install flask 
 pip install sentence-transformers 
 pip install spacy 
 pip install numpy 
6. Download Language Models: 
Download the pre-trained models for SBERT and spaCy. Follow the 
documentation for each library to get the necessary models. 
7. Code Implementation: 
Front End – HTML, CSS 
Back End – FLASK 
Here backend code done using Flask and used SBERT and spaCy based 
Summarization models to summarize the text. Follow the documentation 
for every necessary codes. 
8. Frontend Integration: 
Integrate the frontend code (HTML, CSS, JavaScript) with the Flask 
backend. Ensure proper communication between the frontend and backend. 
9. Run the Code: 
Once you run the flask code in Vs code or Pycharm a new port number of 
the local server will be displayed on the terminal click on it and you will 
be redirected to live project.
